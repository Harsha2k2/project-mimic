# Architecture Decision Records

This folder stores Architecture Decision Records (ADRs) for Project Mimic.

## How To Use

- Copy `TEMPLATE.md` into a new numbered file.
- Use ascending sequence numbers: `0001-...`, `0002-...`.
- Keep each ADR focused on one decision.
